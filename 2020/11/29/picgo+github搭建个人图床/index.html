PicGo

1.安装PicGo

Github资源:https://github.com/Molunerfinn/PicGo/releases

注意：mac 系统选择 dmg 下载，windwos 选择 .exe系统，如果不是下载安装包，想看源码的话，可以选择 git clone https://github.com/Molunerfinn/PicGo.git 克隆到本地 ，建议别下最新版会有莫名的bug

2.安装完成后，运行PicGo

进行PicGo和GitHub的配置

注意：仓库名是你自己的GitHub名称，分支我的默认是main,大佬默认的是master，根据你GitHub的名字；token就是之前保存下来的；指定存储路径就是GitHub里的文件夹，不填写就默认根目录；自定义域名是你最后生成的img的链接地址，配置好点击确定，设置为默认图床

3.测试效果

自行上传图片，上传成功会在右下角提示上传成功，成功后可在GitHub的img文件夹和PicGo的相册都能看见
